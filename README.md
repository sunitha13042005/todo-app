# To-Do List Web Application
A modern To-Do List web application developed using HTML, CSS, and JavaScript to help users manage daily tasks efficiently.

The application allows users to add, delete, search, and mark tasks as completed. Tasks are stored using localStorage so they remain saved even after refreshing the browser.

This project demonstrates strong understanding of frontend development concepts such as DOM manipulation, event handling, and responsive UI design.
## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Search tasks instantly
- Task progress bar
- Task counter
- Tasks saved using LocalStorage
- Responsive and modern UI
- Animated background and glowing card design

 ## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage
- GitHub Pages
